library/hahaha_laravel_lib Tests 說明

命名備註

- 測試檔名與測試 class 採專案風格，並補 PHPUnit 可辨識後綴：`hahaha_test_xxx_xxx_xxx_Test`
- Tests 目錄內若有自訂 public helper method，命名採：`Xxx_Xxx_Xxx(...)`
- PHPUnit 測試方法保留 `test_xxx_xxx_xxx` 命名
- 保留 `test_...` 的原因：目前這批測試直接依 PHPUnit 預設規則辨識，不另外加 `#[Test]` attribute
- 目錄掃描可執行的原因：檔名與 class 名稱已補 `_Test` 後綴，符合 PHPUnit `*Test.php` 掃描規則
- `setUp()`、`tearDown()` 保留 PHPUnit / Laravel 生命週期名稱，不改成其他格式
- 這批測試請用指定檔案路徑執行，不依賴 `phpunit.xml` 額外掃描設定

目前重點

- 自訂 helper method 已改成專案風格
- 測試方法名稱維持 PHPUnit 可直接辨識的 `test_...`
- 若未來要把測試方法也改成 `Xxx_Xxx_Xxx`，需同步補 `#[Test]` attribute 或改其他 PHPUnit 辨識策略
- 目前測試已依照目標檔案分到對應資料夾，查找時可直接對照 `Services`、`Providers`、`Console/Commands`

php artisan test --compact --parallel --processes=2 library/hahaha_laravel_lib/Tests
